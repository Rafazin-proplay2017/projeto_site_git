O que é o Git

Git é um sistema de controle de versão distribuído, usado para registrar o histórico de alterações em arquivos de um projeto. Ele permite que várias pessoas trabalhem no mesmo código de forma organizada.

O que é o GitHub

GitHub é uma plataforma online que hospeda repositórios Git. Ele permite colaboração, versionamento e integração com ferramentas de desenvolvimento.

Diferenças entre Git e GitHub

  • Git: é a ferramenta de versionamento.
  • GitHub: é a plataforma que hospeda repositórios Git na nuvem.

Principais comandos Linux para Git Bash

pwd        # mostra o diretório atual
ls         # lista arquivos
cd pasta   # entra em uma pasta
mkdir nome # cria uma pasta
rm nome    # remove arquivos ou pastas
git init   # inicia um repositório Git
git status # mostra o estado do repositório
git add .  # adiciona alterações
git commit -m "mensagem" # salva alterações

Como criar um repositório no GitHub

No GitHub, clique em New Repository, dê um nome, descrição, escolha se será público ou privado e clique em Create Repository.

Como ligar em um projeto existente

git init
git remote add origin https://github.com/seuusuario/repositorio.git
git pull origin main

Como gerar uma versão com Git

git add .
git commit -m "descrição da versão"

Como enviar da máquina para o GitHub

git push origin main
